# Bot Task Briefs

Improvement briefs for `/improve-workflow` and `/full-workflow`.

## Naming

`{slug}-improvements-brief.md` — e.g. `comment-reliability-improvements-brief.md`

## Template

Copy `doc/bot-tasks/_improve-template.md` when scope is non-trivial (multi-file, new feature, or site-wide behavior change).

## Commands

| Command | When to save a brief |
|---------|---------------------|
| `/improve-workflow` | Multi-module fixes, new behavior, or unclear scope |
| `/full-workflow` | Always for new features |

## Status values

- `draft` — planning
- `in-progress` — implementation started
- `done` — verified and complete
